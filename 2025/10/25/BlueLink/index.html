Project：BlueLink Blue Bond RWA Tokenisation

2025-10-25 project

Project Overview

BlueLink is a blue bond fundraising platform built on the Sui blockchain, dedicated to providing transparent and trustworthy funding solutions for ocean conservation, sustainable fisheries, and other blue economy initiatives.

Core Features

Blue Bonds are specialized financial instruments designed to raise capital for ocean conservation and sustainable marine economy projects. On BlueLink, investors can purchase bond tokens to support ocean-related projects and receive principal and interest returns at maturity; issuers can conveniently issue bonds for project fundraising and manage funds through the platform.

Investor Features:

  • Sign in using Sui wallet signature verification for a password-free experience
  • Browse all listed blue bonds with complete information including issuers, interest rates, and maturity dates
  • Purchase bond tokens to support ocean conservation projects and receive NFT bond certificates
  • Track personal bond holdings and earnings status
  • Redeem principal and interest on maturity dates
  • Multi-device management and login device control

Issuer Features:

  • Issue blue bonds to raise funds for ocean projects
  • Manage collected funds and redemption pools
  • Monitor fundraising progress and investor information in real-time
  • Pause/resume bond sales and flexibly control project status

Technical Architecture

Layer Components
Blockchain Layer Sui Move Smart Contracts, Event Listening, RPC API
Backend Go 1.24 + Gin Framework, PostgreSQL Database
Authentication Wallet Signature Verification + Memory Session Management
Frontend (Planned) React + TypeScript + Vite

The system implements real-time data synchronization through blockchain event listening, supporting core events such as BondProjectCreated, BondTokensPurchased, and BondTokenRedeemed to ensure the platform remains synchronized with on-chain state.

Motivation

I developed BlueLink out of a deep understanding of the following challenges:

Current Challenges:

  1. Insufficient Fund Transparency - Traditional ocean conservation fundraising lacks transparent fund tracking mechanisms
  2. High Trust Costs - Investors struggle to verify actual fund usage
  3. High Barriers to Entry - Lack of platformized blue bond issuance tools
  4. Cross-Border Collaboration Difficulties - International investors find it hard to participate in ocean project financing

Blockchain Solutions:

Sui blockchain’s high performance and low-cost characteristics enable us to build:

  • Fully On-Chain - All transaction records are permanently stored and immutable
  • Transparent and Traceable - Fund flows are completely transparent, allowing investors to verify in real-time
  • Efficient and Convenient - Sui’s high throughput delivers second-level confirmations and low transaction fees
  • No Intermediaries Needed - Smart contracts execute automatically, reducing trust costs

Through BlueLink, we aim to connect global investors with ocean conservation projects and promote sustainable development of the blue economy.

Unique Features

1. Fully On-Chain Bond Lifecycle Management

  • Automated Execution - Smart contracts ensure redemption, interest distribution, and other processes execute automatically without manual intervention
  • Immutable Transaction Records - All investment, redemption, and fund management operations are permanently stored on the blockchain
  • Real-Time Synchronization - The system listens to on-chain events every 15 seconds, automatically updating platform data with zero latency

2. Passwordless Authentication Experience

  • Native Sui Signature Verification - Users sign in using Sui wallet signatures without setting or managing complex passwords
  • Replay Attack Prevention - Employs Nonce mechanism to ensure login security
  • Multi-Device Management - View all login devices and revoke suspicious connections anytime (maximum 3 concurrent logins)
  • HttpOnly Cookie - Sessions use HttpOnly mechanism with 24-hour validity period

3. Purpose-Built for Ocean Conservation

  • Blue Economy Specialized Design - Platform specifically designed for ocean-related projects with support for specific fundraising parameters and progress tracking
  • Issuer-Friendly - Project teams can easily issue bonds, manage fundraising, and monitor investor status
  • Investor Transparency - Complete project information display, interest rates, and maturity dates to support risk assessment

4. Enterprise-Grade Security and Reliability

  • Rate Limiting Protection - API rate limiting prevents brute force attacks and DDoS
  • CORS Control - Strict cross-origin request configuration to protect user safety
  • Audit Logs - Request ID tracking and comprehensive audit records for easy fault tracing
  • Soft Delete Mechanism - Preserves data history for traceability and compliance

Future Directions and Ambition

Short-Term Goals (3-6 months)

  1. Frontend Application Launch - Complete React + TypeScript frontend development to provide a complete user interface
  2. Mainnet Deployment - Migrate the platform to Sui mainnet to support real fund flows
  3. Ecosystem Expansion - Support more blue economy project types (aquaculture, ocean cleanup, coral protection, etc.)

Medium-Term Goals (6-12 months)

  1. Cross-Chain Integration - Support blue bonds across multiple public chains to expand investor base
  2. DAO Governance - Establish decentralized governance model to enable community participation in platform decisions
  3. Analytics Dashboard - Provide issuers with detailed analysis reports and investor insights

Long-Term Vision

Advancing United Nations Sustainable Development Goal 14

BlueLink aspires to become the foundational infrastructure for global blue economy financing. Our ultimate objectives are:

  • Global Investor Connection - Connect investors worldwide with ocean conservation projects
  • Enhanced Capital Flow - Dramatically increase accessibility of financing for ocean-related projects
  • Accelerated Ecosystem Protection - Accelerate ocean conservation and sustainable fishery development through adequate funding
  • Transparent Mechanism Establishment - Build industry-standard transparent disclosure and fund tracking mechanisms

I believe that combining blockchain technology with ocean conservation can create a more transparent, efficient, and trustworthy blue economy ecosystem. BlueLink is not just a fundraising platform—it’s a catalyst for driving global sustainable ocean development.
